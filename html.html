Understanding HTML

HTML (HyperText Markup Language) is the backbone of web development. It structures web content and defines elements such as headings, paragraphs, links, and images.

Basic HTML Elements

  • <h1> - <h6>: Headings
  • <p>: Paragraphs
  • <a>: Hyperlinks
  • <img>: Images
  • <table>: Tables
  • <div> & <span>: Layout elements
